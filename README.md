# Aptos-Perps-Yields-Decibel
Aptos Décibel Banger
Projet où se placer early la phase de test n'a pas encore commencé. 
Le projet c'est décibel, un projet sur aptos qui compte créer un perps avec certaines spécificité.
De ce que je comprends la spécificité se fait au niveau du moteur trading utilisé qui unifiera le spot, le perpétuels et la yield sur Aptos.
Pour le moment on est en phase de développement, le testnet est pour Q4 2025 et l'application pour Q1 2026. 
Decibel est en collab avec Aptos Labs, au début d'année la Aptos Foundation a levé 200 M de $ pour investir dans les boîtes defi machinant sur Aptos.
